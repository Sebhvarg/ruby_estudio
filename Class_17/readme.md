# Ruby Calculator

## Overview
A basic command-line calculator program written in Ruby that performs arithmetic operations.

## Current State
- Fully functional calculator with basic operations
- Clean command-line interface
- Error handling for invalid inputs and division by zero

## Features
- **Addition** (+): Add two numbers
- **Subtraction** (-): Subtract second number from first
- **Multiplication** (*): Multiply two numbers
- **Division** (/): Divide first number by second (with zero-division protection)

## How to Run
The calculator runs in a continuous loop. Enter numbers and operators when prompted.
Type 'exit', 'quit', or 'q' to close the program.
